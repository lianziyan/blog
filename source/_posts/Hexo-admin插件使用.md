## 1.下载依赖包

``` node
npm install --save hexo-admin
```

## 2.启动hexo服务
``` node
npm server -d
```

## 3.修改用户名、密码
  在Hexo博客控制台页面->设置->用户名、密码一一填写，填写完毕后将页面最下面的config.yaml配置一键复制、粘贴，重启后即可生效
